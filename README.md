# 🏠 House Price Prediction (ML Project)

## 📌 Overview

This project predicts house prices based on the square footage of a house using a simple machine learning model.

---

## 🎯 Problem Statement

Estimate the price of a house using only its area (in square feet).

---

## ⚙️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

---

## 🧠 Model Used

* Linear Regression

---

## 🔄 How It Works

1. Data is collected (square feet vs price)
2. Data is cleaned and prepared
3. Model is trained using Linear Regression
4. Model is saved using pickle
5. User inputs square feet → model predicts price

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python src/api.py
```

---

## 📥 Input

* Square feet (e.g., 1200)

## 📤 Output

* Predicted house price
